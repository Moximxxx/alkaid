## 经验教训

### 1. Electron 主进程代理模式

**场景**：前端调用第三方 AI API 存在 CORS 限制

**解决方案**：
```
渲染进程 (fetch localhost:3000) → 主进程代理 → AI Provider API
```

**优势**：
- 主进程不受 CORS 限制
- 可统一管理 API Key
- 便于日志记录和错误处理

**实现要点**：
- 使用 Node.js `http.createServer` 创建代理
- 流式响应需正确处理 `for await...of` 迭代
- 注意 `stream: true` 时的 `text/event-stream` 头部设置

### 2. Electron 环境判断

**正确写法**（main.cjs 第 75、99 行）：
```javascript
const isDev = process.env.NODE_ENV === 'development' || !app.isPackaged;
```

**用途**：
- 开发环境：加载 `localhost:5173`
- 生产环境：加载 `../dist/index.html`
